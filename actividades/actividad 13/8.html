Letter and Word Spacing

by TPS SENA

El kerning es el término que utilizan los tipógrafos para el espcio entre cada letra. Puede controlar el espacio entre cada letra con la propiedad: Letter-spacing. Es particularmente útil para aumentar el kerning cuando su encabezado o frase está en mayúsculas. Si el texto está en un caso o normal, aumentar o disminuir el kerning puede dificultar la lectura.

El intervalo predeterminado

Cuando se especifica un valor para estas propiedades, se debe dar en EMS. El intervalo predeterminado entre las palabras se establece por el tipo de letra (a menudo alrededor de 0,25 EM), y es poco probable que deba cambiar esta propiedad con regularidad.
