Controlando la posición de los elementos

Esquemas de posicicionamiento

CSS tiene esquemas de posicionamiento que le permiten controlar el diseño de una página: Normal Flow (Lo vimos anteriormente), Relative Positioning y Absolute Positioning. Especifique el esquema de posicionamiento utilizando la propiedad: position en CSS. También puede ubicar elementos mediante la propioedad Float

Relative Positioning

Este posicicionamiento mueve un elemento en relación con el lugar donde habría estado en Normal Flow, desplazándo el elemento a la parte superior, derecha, inferior o izquierda de donde se habría colocado. Esto no afecta la posición de los elementos circundantes. los valores de las propiedades de desfase de cuadro se suelen dar en píxeles, porcentajes o ems.
