VOLVER


El elemento <div> permite agrupar un conjunto de elementos juntos en un cuadro de nivel de bloque. Por ejemplo, puede crear un elemento <div> para que contenga todos lo elementos del encabezado del sitio (el logotipo y la navegación), o puede crear un elemento <div> para que contenga información en bloques

En un navegador, el contenido del elemento <div> comenzará en un nueva línea, pero aparte de esto no hará ninguna diferencia en la presentanción de la página. Sin embargo, el uso de un atributo ID o Class en el elemento <div> dignifica que puede crear reglas de estilo CSS para indicar cuánto espacio debe ocupar el elemento <div> en la pantalla y cambiar la apariencia de todos los elementos contenidos en el. También puede facilitar el seguimiento del código si ha utilizado elementos <div> para contener cada sección de la página

Dado que puede haber varios otros elementos dentro de un elemento <div>, puede ser útil agregar un comentario depués de la etiqueta </div> de cierre. Esto le permite ver claramente a qué etiqueta de apertura se supone que corresponde.
