Background Color - Color de Fondo

Si no se espesifica un color de fondo, el fondo es transparente.

CSS trata cada elemento HTML como un cuadro y la propiedad Backgorund-color establece el color del fondo para el cuadro. Se puede especificar un color de fondo de las mismas tres formas en que puede especificar color de primer plano: Valores RGB, Código Hexadecimal y Nombre de Colores. también hemos utilizado la propiedad padding para separar el texto de los bordes de los cuadros. Esto hace que se más facil de leer.
