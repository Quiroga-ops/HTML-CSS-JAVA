Cada elemento HTML puede llevar el atributo id. Se utiliza para identidicar de forma única ese elemento de otros elementos de la pagina. Su valor debe comenzar con una letra o un guión bajo (no un número o cualquier otro carácter). Es importante que no hay dos elementos en la misma página tienen el mismo valor para sus atribuos ID (de lo contrario, el valor ya no es único)

dar a un elemento una identidad única le permite darle un estilo diferente que cualquier otras instancia del mismo elemento de la pagina.

Por ejempli, es posible que desee asignar un párrafo dentro de la página (quizás un párrafo que contiene una cotizacion de extracción) un estilo diferente que todos los demás párrafos. En el ejemplo, el párrafo con el atributo ID cuyo valor es pullquote se hace en matúsculas usando CSS.
